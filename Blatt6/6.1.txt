Exercise 1 – Node.js Basic Concepts

a) Explain the relation between JavaScript and Node.js.

		Node.js sorgt dafür das JavaScript-Code außerhalb des Browsers auf der Serverseite laufen kann.

b) Why is it a golden rule of Node.js not to block the event queue?

		Wenn die event queue geblockt wird kommt der node.js server zum stoppen.

c) What is Express? What does it offer and what is it used for?

		Webanwendungen
			Express ist ein einfaches und flexibles Node.js-Framework von Webanwendungen, 
			das zahlreiche leistungsfähige Features und Funktionen für Webanwendungen und mobile Anwendungen bereitstellt.
		
		APIs
			Mithilfe unzähliger HTTP-Dienstprogrammmethoden und Middlewarefunktionen 
			gestaltet sich das Erstellen einer leistungsfähigen API schnell und einfach.
			
		Leistung
			Express bietet eine Thin-Layer-Ebene mit grundlegenden Webanwendungsfunktionen, 
			ohne die bekannten Node.js-Features zu überlagern.

d) Explain the concept of Routes in Express.

		Per Routing wird bestimmt, wie eine Antwort auf eine Clientanforderung an einem bestimmten Endpunkt antwortet. 
		Dies ist eine URI (oder ein Pfad) und eine bestimmte HTTP-Anforderungsmethode (GET, POST usw.).
 
		/about =auf den pfad mit about
		/ab?cd = auf abcd oder acd weil vor dem ? Ist das Zeichen optional
		/abcd = abcd, acd, abbbbbbbcd, das b kann beliebig oft vorkommen
		/.fly$ = $ heißt der string muss auf fly enden er kann allem anfangen " ." Heißt alles mögliche * beliebig oft
		/ab+cd = abcd, abbbbbcd wieder beliebig oft vorkommen, mind 1 mal
		/ab(cd)?e = abcee, abe… ? Bezieht sich somit auf die ganze klammer
		 
		 
		Route Parameter:
		 
		: heißt alles ab : bis zum ende oder / ist ein parameter
		 
		/users/:userId/books/:bookId
		Wird zu 
		/users/34/books/8989
		Mit parameter
		{"userId": "34", "bookId": "8989"}
